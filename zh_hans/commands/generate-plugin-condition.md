# generate:plugin:condition
生成条件插件

**用法:**
```
$ drupal generate:plugin:condition [options]
```

## 可用选项
选项 | 详细
-------|-------------
--module | 模块名称
--class | 条件插件类名
--label | 条件插件标签
--plugin-id | 条件插件 ID
--context-definition-id | 上下文定义 ID
--context-definition-label | 上下文定义标签
--context-definition-required | 上下文定义是必须的（TURE 或 FALSE）
