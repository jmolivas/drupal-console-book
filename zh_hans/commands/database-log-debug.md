# database:log:debug
显示网站日志事件(s)

**用法:**
```
$ drupal database:log:debug [arguments] [options]
```

## 可用选项
选项 | 详细
-------|-------------
--type | 根据类型过滤事件
--severity | 根据严重等级过滤事件
--user-id | 根据用户 ID 过滤事件
--reverse | 反序输出事件
--limit | 限制显示结果的数量
--offset | 偏移量

## 可用参数
参数 | 详细
---------|-------------
event-id | DBLog 事件 ID
