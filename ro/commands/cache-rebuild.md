# cache:rebuild
Reconstruiți și curățați tot cache-ul.

**Folosire:**
```
$ drupal cache:rebuild [arguments]
$ cr  
```

## Argumente disponibile
Argument | Detalii
---------|-------------
cache | Curățați doar cache-ul specificat.

## Exemple
* Reconstruiți tot cache-ul
```
$ drupal cr all
```
* Reconstruiți discovery cache
```
$ drupal cr discovery
```
