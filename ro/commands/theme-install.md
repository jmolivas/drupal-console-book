# theme:install
Instalează tema sau temele în aplicație

**Folosire:**
```
$ drupal theme:install [arguments] [options]
$ ti  
```

## Opțiuni disponibile
Opțiune | Detalii
-------|-------------
--set-default | Setați tema ca și temă implicită

## Argumente disponibile
Argument | Detalii
---------|-------------
theme | commands.theme.install.options.module
