# rest:debug
Afişează resursele REST curente.

**Folosire:**
```
$ drupal rest:debug [arguments] [options]
$ rede  
```

## Opțiuni disponibile
Opțiune | Detalii
-------|-------------
--authorization | Starea resursei REST activat | dezactivat

## Argumente disponibile
Argument | Detalii
---------|-------------
resource-id | ID-ul resursei
