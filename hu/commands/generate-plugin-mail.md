# generate:plugin:mail
Generate a plugin mail

**Használat:**
```
$ drupal generate:plugin:mail [options]
```

## Rendelkezésre álló beállítások
Beállítás | Részletek
-------|-------------
--module | A modul neve.
--class | Plugin class name
--label | Plugin label
--plugin-id | Plugin id
--services | Szolgáltatások betöltése a tárolóból.
