# test:debug
Az alkalmazás egységtesztjeinek listázása.

**Használat:**
```
$ drupal test:debug [arguments] [options]
$ td  
```

## Rendelkezésre álló beállítások
Beállítás | Részletek
-------|-------------
--test-class | Tesztosztály

## Rendelkezésre álló argumentumok
Argumentum | Részletek
---------|-------------
group | Csoport
