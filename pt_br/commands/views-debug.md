# views:debug
Exibir informações sobre a View atual

**Uso:**
```
$ drupal views:debug [arguments] [options]
$ vde  
```

## Opções disponíveis
Opção | Detalhes
-------|-------------
--tag | Tag da View
--status | Status da View (Habilitado|Desabilitado)

## Argumentos disponíveis
Argumento | Detalhes
---------|-------------
view-id | ID da View
