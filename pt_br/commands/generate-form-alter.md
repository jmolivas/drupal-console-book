# generate:form:alter
Gera uma implementação de hook_form_alter() ou hook_form_FORM_ID_alter

**Uso:**
```
$ drupal generate:form:alter [options]
$ gfa  
```

## Opções disponíveis
Opção | Detalhes
-------|-------------
--module | O nome do módulo.
--form-id | Form ID para alterar
--inputs | Cria entradas de formulário.
