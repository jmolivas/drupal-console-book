# test:debug
Listar testes unitários disponíveis para a aplicação.

**Uso:**
```
$ drupal test:debug [arguments] [options]
$ td  
```

## Opções disponíveis
Opção | Detalhes
-------|-------------
--test-class | Classe do teste

## Argumentos disponíveis
Argumento | Detalhes
---------|-------------
group | Grupo
