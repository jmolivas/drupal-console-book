# rest:debug
Display current rest resource for the application

**Uso:**
```
$ drupal rest:debug [arguments] [options]
$ rede  
```

## Opções disponíveis
Opção | Detalhes
-------|-------------
--authorization | Rest resource status enabled | disabled

## Argumentos disponíveis
Argumento | Detalhes
---------|-------------
resource-id | Rest ID
