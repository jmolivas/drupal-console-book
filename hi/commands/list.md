# list
कमांड की सूची

**प्रयोग:**
```
$ drupal list [arguments] [options]
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--xml | XML के रूप में उत्पादन सूची
--raw | रॉ कमांड की उत्पादन सूची
--format | आउटपुट फॉर्मेट (txt, xml, json, or md)

## उपलब्ध तर्कों
तर्क | विवरण
---------|-------------
namespace | नेमस्पेस का नाम
