# theme:install
इंस्टॉल विषय या विषयों के आवेदन में

**प्रयोग:**
```
$ drupal theme:install [arguments] [options]
$ ti  
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--set-default | डिफ़ॉल्ट विषय के रूप में विषय सेट करे

## उपलब्ध तर्कों
तर्क | विवरण
---------|-------------
theme | commands.theme.install.options.module
