# views:debug
अनुप्रयोग के वर्तमान व्यूज साधनो को दिखाएँ

**प्रयोग:**
```
$ drupal views:debug [arguments] [options]
$ vde  
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--tag | व्यू टैग
--status | व्यू स्टेटस (Enabled|Disabled)

## उपलब्ध तर्कों
तर्क | विवरण
---------|-------------
view-id | व्यू आईडी
