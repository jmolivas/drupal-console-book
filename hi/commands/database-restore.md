# database:restore
MYSQL डाटाबेस और टेबल्स ले कंटेंट और संरचना को रिस्टोर करे

**प्रयोग:**
```
$ drupal database:restore [arguments] [options]
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--file | अपने डेटाबेस बैकअप फ़ाइल के लिए फ़ाइल नाम

## उपलब्ध तर्कों
तर्क | विवरण
---------|-------------
database | Settings.php से डाटाबेस कुंजी
