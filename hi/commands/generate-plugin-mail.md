# generate:plugin:mail
Generate a plugin mail

**प्रयोग:**
```
$ drupal generate:plugin:mail [options]
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--module | मोड्यूल का नाम।
--class | Plugin class name
--label | Plugin label
--plugin-id | Plugin id
--services | सर्विसेज़ को container से लोड करें।
