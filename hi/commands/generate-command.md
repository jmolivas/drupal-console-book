# generate:command
कंसोल के लिए कमाण्डो को उत्पन्न करें।

**प्रयोग:**
```
$ drupal generate:command [options]
$ gcm  
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--module | मोड्यूल का नाम।
--class | कमाण्ड क्लास का नाम
--name | कमाण्ड का नाम।
--container-aware | Drupal साइट स्थापना के बारे में कमांड पता है जब एक्सेक्युट होगा?
