# help
एक कमांड के लिए मदद प्रदर्शित करता है

**प्रयोग:**
```
$ drupal help [arguments] [options]
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--xml | XML के रूप में उत्पादन सूची
--raw | रॉ कमांड सूची उत्पादन करने के लिए
--format | फॉर्मेट आउटपुट (txt, xml, json, or md)

## उपलब्ध तर्कों
तर्क | विवरण
---------|-------------
command_name | कमांड का नाम
