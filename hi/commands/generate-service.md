# generate:service
सर्विस उत्पन्न करें

**प्रयोग:**
```
$ drupal generate:service [options]
$ gs  
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--module | मोड्यूल का नाम।
--name | commands.generate.service.options.name
--class | क्लास का नाम
--interface | commands.common.service.options.interface
--services | सर्विसेज़ को container से लोड करें।
