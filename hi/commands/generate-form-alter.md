# generate:form:alter
एक hook_form_alter() या hook_form_FORM_ID_alter कार्यान्वयन उत्पन्न करें

**प्रयोग:**
```
$ drupal generate:form:alter [options]
$ gfa  
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--module | मोड्यूल का नाम।
--form-id | फॉर्म आईड को बदलने के लिए
--inputs | एक फॉर्म के लिए आदानो को बनाएँ।
