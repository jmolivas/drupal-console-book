# create:vocabularies
अपने Drupal 8 एप्लीकेशन के लिए डमी वोकैब्युलरीस बनाएँ।

**प्रयोग:**
```
$ drupal create:vocabularies [options]
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--limit | आप कितने वोकैब्युलरीस बनाना चाहते है
--name-words | वोकैब्युलरी के नाम में शब्दों की अधिकतम संख्या
