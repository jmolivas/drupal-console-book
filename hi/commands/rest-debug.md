# rest:debug
आवेदन के लिए मौजूदा REST संसाधन को प्रदर्शित करें

**प्रयोग:**
```
$ drupal rest:debug [arguments] [options]
$ rede  
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--authorization | REST साधन की स्थिति (चालू है । बंद है )

## उपलब्ध तर्कों
तर्क | विवरण
---------|-------------
resource-id | Rest क्रमांक
