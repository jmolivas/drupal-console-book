# database:dump
डंप संरचना और MYSQL का कंटेंट डेटाबेसेस और टेबल्स

**प्रयोग:**
```
$ drupal database:dump [arguments] [options]
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--file | commands.database.dump.option.file

## उपलब्ध तर्कों
तर्क | विवरण
---------|-------------
database | Settings.php से डाटाबेस कुंजी
