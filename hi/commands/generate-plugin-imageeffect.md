# generate:plugin:imageeffect
छवि प्रभाव प्लगिन उत्पन्न करें

**प्रयोग:**
```
$ drupal generate:plugin:imageeffect [options]
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--module | मोड्यूल का नाम।
--class | प्लगिन का क्लास नाम
--label | प्लगिन उप-शीर्षक
--plugin-id | प्लगिन id
--description | प्लगिन विवरण
