# rest:debug
Hiển thị rest resource hiện có cho ứng dụng

**Usage:**
```
$ drupal rest:debug [arguments] [options]
$ rede  
```

## Các tùy chọn có sẵn
Tùy chọn | Các chi tiết
-------|-------------
--authorization |  Trạng thái rest resource bật | tắt

## Các đối số có sẵn
Đối số | Các chi tiết
---------|-------------
resource-id | Rest ID
