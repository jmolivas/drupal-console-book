# generate:form:alter
Tạo một implementation của hook_form_alter() hoặc hook_form_FORM_ID_alter

**Usage:**
```
$ drupal generate:form:alter [options]
$ gfa  
```

## Các tùy chọn có sẵn
Tùy chọn | Các chi tiết
-------|-------------
--module | Tên module.
--form-id | Từ ID đến alter
--inputs | Tạo các đầu vào trong một form.
