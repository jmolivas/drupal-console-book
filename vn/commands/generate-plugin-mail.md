# generate:plugin:mail
Generate a plugin mail

**Usage:**
```
$ drupal generate:plugin:mail [options]
```

## Các tùy chọn có sẵn
Tùy chọn | Các chi tiết
-------|-------------
--module | Tên module.
--class | Plugin class name
--label | Plugin label
--plugin-id | Plugin id
--services | Nạp các dịch vụ từ container.
