# theme:install
Cài đặt theme hoặc các themes trong ứng dụng

**Usage:**
```
$ drupal theme:install [arguments] [options]
$ ti  
```

## Các tùy chọn có sẵn
Tùy chọn | Các chi tiết
-------|-------------
--set-default | Thiết lập theme thành theme mặc định

## Các đối số có sẵn
Đối số | Các chi tiết
---------|-------------
theme | commands.theme.install.options.module
