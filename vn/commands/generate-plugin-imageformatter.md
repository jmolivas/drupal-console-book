# generate:plugin:imageformatter
Phát sinh một image formatter plugin.

**Usage:**
```
$ drupal generate:plugin:imageformatter [options]
```

## Các tùy chọn có sẵn
Tùy chọn | Các chi tiết
-------|-------------
--module | Tên module.
--class | Tên plugin class
--label | Nhãn plugin
--plugin-id | Plugin id
