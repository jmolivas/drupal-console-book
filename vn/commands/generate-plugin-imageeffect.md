# generate:plugin:imageeffect
Tạo plugin hiệu ứng hình ảnh

**Usage:**
```
$ drupal generate:plugin:imageeffect [options]
```

## Các tùy chọn có sẵn
Tùy chọn | Các chi tiết
-------|-------------
--module | Tên module.
--class | Tên lớp plugin
--label | Nhãn plugin
--plugin-id | Plugin id
--description | Mô tả plugin
