# generate:authentication:provider
Tạo ra một Authentication Provider

**Usage:**
```
$ drupal generate:authentication:provider [options]
$ gap  
```

## Các tùy chọn có sẵn
Tùy chọn | Các chi tiết
-------|-------------
--module | Tên module.
--class | Lớp Authentication Provider
--provider-id | Provider ID
