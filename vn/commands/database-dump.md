# database:dump
Dump cấu trúc và nội dung của 1 cơ sở dữ liệu

**Usage:**
```
$ drupal database:dump [arguments] [options]
```

## Các tùy chọn có sẵn
Tùy chọn | Các chi tiết
-------|-------------
--file | commands.database.dump.option.file

## Các đối số có sẵn
Đối số | Các chi tiết
---------|-------------
database | Database key từ settings.php
