# generate:service
Tạo dịch vụ

**Usage:**
```
$ drupal generate:service [options]
$ gs  
```

## Các tùy chọn có sẵn
Tùy chọn | Các chi tiết
-------|-------------
--module | Tên module.
--name | commands.generate.service.options.name
--class | Tên Class
--interface | commands.common.service.options.interface
--services | Nạp các dịch vụ từ container.
