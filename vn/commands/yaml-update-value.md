# yaml:update:value
Cập nhật một giá trị cho một key cụ thể trong file YAML.

**Usage:**
```
$ drupal yaml:update:value [arguments]
$ yuv  
```

## Các đối số có sẵn
Đối số | Các chi tiết
---------|-------------
yaml-file | Đường dẫn của file YAML để cập nhật
yaml-key | Khóa YAML để cập nhật
yaml-value | Giá trị YAML để cập nhật
