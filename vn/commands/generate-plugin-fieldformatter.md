# generate:plugin:fieldformatter
Tạo một formatter plugin.

**Usage:**
```
$ drupal generate:plugin:fieldformatter [options]
```

## Các tùy chọn có sẵn
Tùy chọn | Các chi tiết
-------|-------------
--module | Tên module.
--class | Tên lớp plugin
--label | Nhãn plugin
--plugin-id | Plugin id
--field-type | Loại field mà plugin có thể được sử dụng với
