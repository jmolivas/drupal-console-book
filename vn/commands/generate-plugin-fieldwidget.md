# generate:plugin:fieldwidget
Tạo field widget plugin.

**Usage:**
```
$ drupal generate:plugin:fieldwidget [options]
```

## Các tùy chọn có sẵn
Tùy chọn | Các chi tiết
-------|-------------
--module | Tên module.
--class | Tên lớp plugin
--label | Nhãn plugin
--plugin-id | Plugin id
--field-type | Kiểu plugin có thể được sử dụng với
